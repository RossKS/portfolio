Firstview Lettings Ltd

Website built using:

  • HTML
  • CSS/Sass
  • JavaScript/JQuery
  • Bootstrap 4
  • PHP
  • Git/GitHub
View Live Site View GitHub Repo

For Firstview Lettings Ltd I created a website that reflected their simplicity with an easy to use UX and informative copy. The goal for the site was to create a modern website with increased usability and clarity of their product offering. Overall the business wanted to attract and convert more visitors.

I developed it mostly using CSS but used Sass for some mixins and variables. It uses a small amount of Bootstrap4, a JQuery carousel animation and I coded the contact form using PHP which sanitizes and validates user input. It's designed mobile first and works on all screen sizes which is crucial for modern websites.
